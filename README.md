# Surge

本项目致力于用最精简的方式去除大部分软件的广告影响体验，如开屏弹窗、横幅等。

由于本人使用 Surge，若使用其他软件请通过 [Script-Hub](https://github.com/Script-Hub-Org/Script-Hub) 自行转换。

喜欢的话请帮忙点下 🌟🌟
